#### var 和 let const 的区别

- var是ES5语法，let const 是ES6语法；var有变量提升
- var 和 let 是变量，可修改；const是常量，不可修改
- let const 有块级作用域，var 没有

#### typeof 返回那些数据类型

- number、string、boolean、undefined、symbol
- object(注意 typeof null === 'object')
- function

#### 列举强制类型转换和隐式类型转换

- 强制：parseInt、parseFloat、toString等
- 隐式：if、逻辑运算、==、 + 拼接字符串
